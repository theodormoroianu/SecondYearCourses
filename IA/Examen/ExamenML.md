## Formule

```
D_1((a, b), (x, y)) = rad_1(abs(a - x)^1 + abs(b - y)^1)
D_2((a, b), (x, y)) = rad_2(abs(a - x) ^ 2 + abs(b - y) ^ 2)
```

```
Precision           = TP/(TP+FP)
Sensitivity(recall) = TP/(TP+FN)
Specificity         = TN/(TN+FP)
Accuracy            = (TP+TN)/(TP+TN+FP+FN)
F1                  = 2/(1/Precision+1/Recall)
```
